Characters used for markup in Markdown and/or GCW.

Normal: []{}()#+-.!

Escaped: \\, \`, \*, \_, \[, \], \{, \}, \(, \), \#, \+, \-, \., \!

In a code block:

    Code block
    \`*_[]{}()#+-.!
    \\, \`, \*, \_, ,\[, \], \{, \}, \(, \), \#, \+, \-, \., \!

- monospace normal ``\`*_[]{}()#+-.!``
- monospace escaped ``\\, \`, \*, \_, ,\[, \], \{, \}, \(, \), \#, \+, \-, \., \!``

GCW code markers: {{{ foo }}} bar \`{{{\`}}}
